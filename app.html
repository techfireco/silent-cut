Serve from a local server — not file://

Browser security blocks FFmpeg.wasm when opened directly as a file. Run one of these commands in the same folder as the HTML file:

npx serve . python3 -m http.server 8080

Then open http://localhost:8080 (or the URL shown in terminal).
