Changelog

A complete history of updates, improvements, and fixes.

v2.0 latest April 14, 2026
  • Interactive timeline with pan (scroll) and zoom (Ctrl+Scroll)
  • Click to select silence regions, Delete/X to remove, double-click to toggle
  • Drag on waveform to manually create cut regions
  • Resize region edges by dragging
  • Full undo support with Ctrl+Z / Cmd+Z
  • Playback engine with Skip Silence toggle for real-time preview
  • Smooth auto-scrolling timeline during playback
  • Timeline ruler with adaptive time markers
  • Keyboard shortcuts: Space (play/pause), Delete, Ctrl+Z
  • Output in same format as input (no forced MP4/MP3 conversion)
  • Format-aware codec selection (VP8 for WebM, PCM for WAV, etc.)
v1.0 April 13, 2026
  • Initial release of AutoCut Studio (now SilentCut)
  • FFmpeg.wasm integration for browser-based video/audio processing
  • Voice-aware silence detection with adjustable threshold, duration, and padding
  • Waveform visualization with voice/silence color coding
  • Support for 10 file formats: MP4, MOV, MKV, WebM, AVI, MP3, WAV, OGG, M4A, FLAC
  • 100% local processing — zero file uploads
  • Patched FFmpeg loader to avoid CORS/Worker issues on any hosting
